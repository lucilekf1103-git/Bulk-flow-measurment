# The project 

These codes were developed during my M2 internship at CPPM for 4 months between March and July 2026. The goal is to calculate the bulk flow and the shear moment for the peculiar velocities of DESI DR1. But the code is adaptable to the use of other catalogs such as CosmicFlow4 Tully-Fisher (CF4TF).
These codes were developed by Fei Qin, myself and with the help of Julian Bautista. You can find above a code calculating the first two moments of peculiar velocity (Bulk flow, shear moment) for the data (data_calc.py) which is taking about 20min to run for 80 000 peculiar velocities, a code calculating these same moments for mocks (mocks_calc.py) taking about 2min for each mocks and a code calculating the theoretical prediction of the LambdaCDM cosmological model (LCDM_prediction.py) that is taking for a subsample of 2500 galaxies around 3h.
The code (results_analysis.ipynb) is a jupiter notebook which, from the matrices calculated by the previous codes, summarizes the values ​​of the calculated moments and performs a chi2 stratistic analysis as well as the associated p-value in order to compare the data with theoretical predictions. I also added the calculation of the direction of the bulk flow in galactical coordinates.
The (BF_OptMC.py) and (CosmoFunc.py) are functions used is the previous codes developed by Fei Qin.
